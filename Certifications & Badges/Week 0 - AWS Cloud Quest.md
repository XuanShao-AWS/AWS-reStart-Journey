## AWS Cloud Quest: Cloud Practitioner - Week 0
a role-based learning experience designed to help you start your cloud career or transition from a non-cloud role.

### Key Features
* **Hands-on Learning:** 12 FREE assignments guiding you through real-world solutions.
* **Interactive Content:** Enriched by instructional videos, architecture diagrams, and knowledge quizzes.
* **Gamified Experience:** Create your avatar, solve virtual customer problems, and upgrade your city.
* **Competitive Edge:** Compete in tournaments and climb the leaderboards.

---

## Objectives

- Build real-world solutions  
- Gain hands-on experience with AWS services  
- Learn fundamental cloud concepts  
- Prepare for AWS certification

## AWS Services

- Amazon EC2 Auto Scaling  
- Amazon CloudWatch  
- Amazon DynamoDB  
- Amazon Elastic Compute Cloud  (EC2)
- Amazon Elastic File System  (EFS)
- Elastic Load Balancing  (ELB)
- AWS Identity and Access Management  (IAM)
- Amazon Relational Database Service  (RDS)
- Amazon Simple Storage Service  (S3)
- Amazon VPC
